## Results & Metrics

### Quantitative Results

| Metric                   | Before | After  | Improvement                |
| ------------------------ | ------ | ------ | -------------------------- |
| Voice bot calls handled  | -      | 4.8M+  | 4.8M+ calls since launch   |
| Containment rate (voice) | -      | 40%    | 40% without live agent     |
| Chatbot interactions     | -      | 450K+  | 450K+ interactions         |
| Chatbot self-service     | -      | 42%    | 42% resolved without agent |
| Investment               | -      | $13M   | $13M deployment cost       |

- Described as "incredible return on investment"

- Payment plan setup automation

## Challenges Noted

- 2024 performance declined (29% self-serve, down from 42%)
- Advisory panel recommends LLM integration
- Bots refer to general information vs. specific answers
