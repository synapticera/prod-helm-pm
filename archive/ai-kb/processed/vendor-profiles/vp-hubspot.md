## How It Works: HubSpot Breeze AI Architecture

### Breeze AI Customer Agent Flow

```
Customer Inquiry (Chat/Email)
        ↓
┌─────────────────────────────────────┐
│  Breeze AI Customer Agent           │
│  • Session initialization           │
│  • Customer identification          │
│  • Context retrieval (CRM)          │
└─────────────────────────────────────┘
        ↓
┌─────────────────────────────────────┐
│  Knowledge Base Search              │
│  • Direct KB connection             │
│  • Article matching                 │
│  • Response synthesis               │
│  • Zero-training required           │
└─────────────────────────────────────┘
        ↓
┌───────────────┬───────────────┬───────────────┐
│  AI Response  │   Ticket      │   Human       │
│  (Self-Serve) │   Creation    │   Handoff     │
└───────────────┴───────────────┴───────────────┘
```

### Sub-1-Hour Deployment Pattern

```
Setup Start
        ↓
┌─────────────────────────────────────┐
│  1. Connect Knowledge Base          │
│     • Point to existing HubSpot KB  │
│     • Auto-ingest content           │
│     • No training required          │
└─────────────────────────────────────┘
        ↓
┌─────────────────────────────────────┐
│  2. Configure Basic Settings        │
│     • Widget placement              │
│     • Escalation rules              │
│     • Operating hours               │
└─────────────────────────────────────┘
        ↓
┌─────────────────────────────────────┐
│  3. Go Live                         │
│     • Deploy to website             │
│     • 24/7 coverage enabled         │
└─────────────────────────────────────┘
        ↓
<1 Hour Total Setup Time
```
