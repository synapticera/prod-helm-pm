Technical Architecture

Core Components:

  • Task Nodes: Every agent execution
  • Evidence Nodes: Docs, APIs, tool outputs
  • Policy Nodes: Rules and compliance
  • Decision Links: Why decisions were made
  • Precedent Links: Similar past cases

Integration Points:

  • Automatic graph writer in execution engine
  • GraphRAG query engine for retrieval
  • Neo4j property graph database
  • Hybrid: Vector search + Graph traversal
