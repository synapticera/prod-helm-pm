Core Filters - Most Used

Essential filters for basic process mining analysis. Start here for common filtering needs.

Cases with Attribute

Select or remove cases based on case-level or event-level attribute values with flexible comparison methods.

Time Period

Filter cases based on date and time criteria using specific dates, relative periods, or date ranges.

Case Start

Select or remove cases based on the attribute value in the first event of each case.

Case End

Filter cases based on the attribute value in the last event of each case.

Activity Not Performed

Find cases where a specified activity was never executed, useful for identifying process deviations.

Activity Performed Once

Select cases where a specified activity was executed exactly one time.

Event Order

Identify cases based on sequential relationships between activities (directly follows, eventually follows, same time/date).

Process Variants

Filter cases by process variant frequency using variant count, case percentage, or variant percentage methods.

Selected Activity Frequency

Identify cases based on how many times a specific activity occurs within each case.

Advanced Filtering

Commonly used filters for detailed process analysis and specific scenarios.

Events with Attribute

Filter individual events based on attribute values at the event level.

Cases with Changed Attribute

Select cases where an event attribute has different values across events within the case.

Value Frequency

Filter cases based on how frequently their attribute values appear across the entire dataset.

Deadline

Select cases based on whether activities were performed before, after, or on the same day as deadline timestamps.

Time Between Activities

Filter cases based on the duration between two specific activities.

Compare Attribute Values

Validate mathematical calculations by comparing computed operations against stored result values.

Activity Frequency

Remove activities that appear too rarely or too frequently across your process cases.

Variant Frequency

Select cases based on how frequently their process variant occurs within the dataset.

Select Variants

Manually select specific process variants to focus analysis on particular activity sequences.

Specialized Filters

Advanced filters for specific scenarios and edge cases.

Cases with Unchanged Attribute

Find cases where an event attribute maintains the same value across all events.

Single Activity Cases

Filter out or select cases that contain only a single event.

Text Search

Search for text patterns across case and event attributes.

Cases with Identical Event Timestamps

Find cases where two activities occurred at exactly the same time.

Cases with Identical Event Dates

Identify cases where two activities occurred on the same calendar date.

Remove Activity with Similar Time

Remove events that occur within a specified time window of another activity.

Remove Activity with Same Date

Remove events that occur on the same date as another specified activity.

Events Before or After Activity

Keep only events that occurred before or after a specified activity.

Remove Activity Loops

Remove consecutive duplicate activities that represent loops in the process.

Keep Events with Same Value as Case Attribute

Filter events where an event attribute matches a case-level attribute value.

Remove Data from Underactive Periods

Filter out cases from time periods with low activity levels.

Or Filter

Combine multiple filters with OR logic - cases matching any filter are included.

Filter List

Combine multiple filters with AND logic - cases must match all filters to be included.
