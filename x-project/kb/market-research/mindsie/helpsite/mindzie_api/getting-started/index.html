Prerequisites

Before getting started with the mindzieAPI, ensure you have:

API Access

  • Enterprise Server or SaaS deployment with API access enabled

Credentials

  • Access token, tenant ID, and project ID from your administrator

Network Access

  • HTTPS connectivity to your mindzie instance
