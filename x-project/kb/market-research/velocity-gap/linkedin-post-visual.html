How to Use These Visuals

These are pixel-perfect LinkedIn visuals. To capture them:

  1. Open this file in Chrome at 100% zoom
  2. Use Chrome DevTools (F12) > click the device toolbar icon
  3. For the hero image: Set viewport to 1200 x 627 and screenshot the card
  4. For carousel slides: Set viewport to 1080 x 1080 and screenshot each card
  5. Or right-click > Inspect any card, then right-click the element > Capture node screenshot

Tip: For the best quality, use "Capture node screenshot" from DevTools Inspector.
