# Learning Review Command

Review and summarize your recent learning sessions.

## Usage

This command helps you review what you've learned across multiple sessions.

## Instructions

When the user runs this command:

1. Look for learning session files in the project
2. Summarize key insights from recent sessions
3. Identify patterns and themes across sessions
4. Suggest areas for deeper exploration

## Output Format

Provide a structured review including:
- **Sessions Reviewed**: Count and date range
- **Key Topics**: Main subjects covered
- **Insights Summary**: Most important takeaways
- **Recommended Next Steps**: What to explore next

## Example

```
## Learning Review

### Sessions Reviewed
3 sessions from Dec 1-12, 2025

### Key Topics
- API Design Patterns
- Error Handling Best Practices
- Testing Strategies

### Top Insights
1. Always validate input at system boundaries
2. Use typed errors for better debugging
3. Write tests before implementation

### Recommended Next Steps
- Deep dive into integration testing
- Explore API versioning strategies
```
